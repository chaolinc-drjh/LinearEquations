# 二元一次方程式圖形練習 (Linear Equations Graphing Practice)

這是一個互動式的網頁教育工具，旨在協助學生與一般使用者練習如何在座標平面上繪製「二元一次方程式」的圖形。使用者只需透過拖曳兩點決定一條直線，系統便會即時驗證答案並給予精美的視覺回饋。

## 功能特色 (Features)

- **互動式實作操作**：在座標平面 (Canvas) 上直接拖曳 A、B 兩點，即時畫出對應的直線。
- **智慧自動出題**：系統會自動隨機產生二元一次方程式（例如 `2x + 3y = 6`），並確保解答一定會落在畫面的整數座標網格內。
- **即時視覺化學習回饋**：
  - **答對**：會顯示成功動畫並增加分數。
  - **答錯**：系統除了扣除或不給分數外，還會立刻在畫面上標明紅色的「正確答案」直線以及座標參考點，強化學習效果。
- **回合制與動態計分**：追蹤使用者的答題進度、正確/錯誤題數以及累計分數，讓枯燥的數學練習具有遊戲化的體驗。
- **智慧提示機制**：系統內建動態提示機制（當遇到瓶頸時，會自動提示點座標帶入方程式的計算結果），協助推導解答。
- **現代化精美 UI/UX**：採用時下流行的深色主題 (Dark Mode)、螢光漸層色調、玻璃擬物化 (Glassmorphism) 以及細膩的微動畫特效；並針對平板與桌面端進行了最佳化的版面配置。

## 使用技術 (Tech Stack)

此專案為純前端實作，輕量且效能極佳，不依賴任何大型框架：
- **HTML5**: 建立清晰的語意化與使用者介面結構。
- **CSS3 (Vanilla)**: 運用 CSS Variables、Flexbox 排版以及 Animations，打造炫麗且流暢的特效。
- **JavaScript (Vanilla JS, ES6+)**: 負責 Canvas 畫布渲染、拖曳事件監聽、自動出題演算法及遊戲狀態管理。

## 檔案結構 (File Structure)

- `index.html`: 主程式入口，定義了左側控制面板與右側座標畫布的佈局。
- `style.css`: 所有的視覺樣式設定，包含色彩系統、排版、響應式設計 (RWD) 與狀態動畫。
- `app.js`: 負責所有的核心邏輯，包含網格系統的度量轉換、隨機題庫生成、答案驗證機制。

## 如何執行 (How to Run)

這是一個靜態網頁專案，不需安裝任何伺服器或 npm 套件。

1. **直接開啟**：找到專案資料夾內的 `index.html`，點擊右鍵選擇使用 Chrome, Edge 或 Safari 等現代瀏覽器開啟即可開始使用。
2. **開發伺服器（推薦）**：若您使用 VS Code，推薦安裝 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 擴充功能，對著 `index.html` 點擊右鍵選擇 "Open with Live Server" 啟動。

## 遊戲規則與操作說明 (How to Play)

1. **觀察題目**：在畫面左側面板查看目前的目標方程式（例如： `x - y = 2`）。
2. **找出座標點**：在腦中計算或是利用紙筆，找出兩組符合方程式的 `(x, y)` 座標解。
3. **拖曳座標點設定直線**：在右側座標網格中，將粉紅色點 (A) 與淺藍色點 (B) 拖曳到您算出的座標位置上。
4. **提交驗證**：確認直線與您心中的答案無誤後，點擊左側面板的「✔️ 提交答案」。
5. **檢討結果**：
   - 答對獲得分數，點擊下一題進入挑戰。
   - 若不幸答錯，請觀察畫面上顯示的**紅色正確解答線**與示範座標點，學習並點擊下一題。
6. 回合結束後，系統會結算您的**總得分與答題正確率**，您可以點擊重新開始挑戰更高的分數！
